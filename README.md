Proje Açıklaması
Bu program, C++ dili kullanılarak ikinci düzeyden bir bilinmeyenli denklemlerin (ax^2 + bx + c = 0) köklerini bulmak amacıyla oluşturulmuştur.

Programın Özellikleri:

Kullanıcıdan a,b ve c katsayılarını alır.
Diskriminant (Delta) programlaması yapar.
Delta saate göre;
İki farklı reel kök varsa hesaplar ve yazdırılır.
Çakışık (tek) kök varsa hesaplar ve yazdırılır.
Reel kök veya kullanıcıyı bilgilendirir.
